Enrichir vos pages avec des Images

Les images sont des éléments essentiels pour rendre vos pages web attrayantes et engageantes. Apprenez à les intégrer correctement en respectant les bonnes pratiques d'accessibilité et de performance.

Engagement visuel

Les images capturent l'attention et rendent le contenu plus mémorable pour vos visiteurs.

Accessibilité

Les attributs alt permettent aux technologies d'assistance de décrire les images aux utilisateurs handicapés.

Performance

Des images optimisées améliorent les temps de chargement et l'expérience utilisateur.

Insertion d'images en HTML

En HTML, les images sont insérées à l'aide de la balise <img>. Cette balise est auto-fermante et nécessite au minimum l'attribut src pour spécifier le chemin de l'image, ainsi que l'attribut alt pour fournir une description alternative de l'image.

Exemple de code
<img src="chemin/vers/image.jpg" alt="Description de l'image">

Attributs courants de la balise <img>

  • src : Spécifie le chemin de l'image à afficher. Peut être une URL relative ou absolue.
  • alt : Fournit une description textuelle de l'image, utile pour l'accessibilité et lorsque l'image ne peut pas être chargée.
  • width et height : Permettent de définir la largeur et la hauteur de l'image en pixels.
  • title : Affiche un texte d'infobulle lorsque l'utilisateur survole l'image.
  • loading : Contrôle le chargement paresseux (lazy loading) de l'image.

Formats d'Image Recommandés

JPEG
JPEG/JPG

Idéal pour les photographies et images complexes avec beaucoup de couleurs. Supporte la compression avec perte.

PNG
PNG

Parfait pour les images avec transparence, logos et graphiques. Compression sans perte.

WebP
WebP

Format moderne offrant une meilleure compression que JPEG et PNG. Supporte l'animation et la transparence.

SVG
SVG

Format vectoriel idéal pour les logos, icônes et illustrations. Redimensionnable sans perte de qualité.

Bonnes Pratiques pour les Images

Optimisation des images

Compressez vos images avant de les mettre en ligne pour réduire les temps de chargement. Utilisez des outils comme ImageOptim ou Squoosh.

Textes alternatifs

Toujours fournir un attribut alt descriptif. Pour les images décoratives, utilisez alt="".

Responsive images

Utilisez srcset et sizes pour fournir différentes versions d'une image selon la taille de l'écran.

Lazy loading

Implémentez le chargement paresseux avec loading="lazy" pour améliorer les performances.

Exemple d'image responsive

<img src="image-400.jpg" 
     srcset="image-400.jpg 400w, 
             image-800.jpg 800w,
             image-1200.jpg 1200w"
     sizes="(max-width: 600px) 400px,
            (max-width: 1200px) 800px,
            1200px"
     alt="Image responsive exemple">
