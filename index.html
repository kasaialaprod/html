Découvrez les Fondements du Web

Le HTML (HyperText Markup Language) est le langage fondamental qui structure toutes les pages web que vous consultez. C'est le point de départ essentiel pour quiconque souhaite créer des sites web.

Structure du contenu

Le HTML organise et structure le contenu d'une page web en utilisant des balises sémantiques.

Accessibilité

Un bon HTML améliore l'accessibilité pour tous les utilisateurs, y compris ceux utilisant des technologies d'assistance.

Base solide

Le HTML constitue la fondation sur laquelle le CSS et le JavaScript viennent s'ajouter.

Qu'est-ce que le HTML ?

Le HTML (HyperText Markup Language) est le langage de balisage standard utilisé pour créer des pages web. Il permet de structurer le contenu d'une page en utilisant des balises pour définir différents éléments tels que les titres, les paragraphes, les liens, les images, et bien plus encore.

Structure de base d'un document HTML

Un document HTML commence par une déclaration de type de document <!DOCTYPE html> suivie de la balise <html> qui englobe tout le contenu de la page. À l'intérieur de cette balise, on trouve deux sections principales : <head> et <body>. La section <head> contient des informations méta, des liens vers des feuilles de style, et le titre de la page, tandis que la section <body> contient le contenu visible de la page.

<!DOCTYPE html>
<html>
<head>
Métadonnées et liens
</head>
<body>
Contenu visible
</body>
</html>
<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Ma Première Page</title> </head> <body> <h1>Bienvenue sur ma page</h1> <p>Ceci est mon premier paragraphe en HTML.</p> </body> </html>

Les Éléments HTML Essentiels

Balises de Titre

Les balises <h1> à <h6> définissent les titres et sous-titres, avec <h1> étant le plus important.

<h1>Titre principal</h1>
Paragraphes

La balise <p> définit un paragraphe de texte.

<p>Ceci est un paragraphe.</p>
Liens

La balise <a> crée des liens hypertextes avec l'attribut href.

<a href="https://exemple.com">Visitez Exemple</a>
Images

La balise <img> insère des images avec les attributs src et alt.

<img src="image.jpg" alt="Description">
Listes

Les balises <ul>, <ol> et <li> créent des listes.

<ul><li>Élément</li></ul>
Conteneurs

Les balises <div> et <span> regroupent et structurent le contenu.

<div>Contenu groupé</div>

Bonnes Pratiques en HTML

HTML Sémantique

Utilisez des balises sémantiques comme <header>, <nav>, <main>, <section>, <article>, et <footer> pour donner du sens à votre structure.

Accessibilité

Pensez à l'accessibilité dès le début. Utilisez les attributs alt pour les images, structurez votre contenu logiquement et utilisez les balises appropriées.

Validation

Validez régulièrement votre code HTML avec le validateur du W3C pour vous assurer qu'il respecte les standards.

Séparation des Préoccupations

Le HTML doit se concentrer uniquement sur la structure du contenu. Utilisez CSS pour le style et JavaScript pour l'interactivité. Cette séparation rend votre code plus maintenable et évolutif.

Évolution du HTML

HTML5, la dernière version majeure du langage, a introduit de nombreuses nouvelles balises sémantiques et APIs qui améliorent à la fois la structure des pages et les possibilités d'interaction. Les balises comme <header>, <nav>, <section>, <article>, et <footer> donnent du sens à la structure de la page, améliorant à la fois l'accessibilité et le référencement.
